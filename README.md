# AIOS — Architecture Decision Records & Design Documents

AIOS（AI Operating System）是一个面向个人的 AI 协作操作系统。本仓库为 AIOS 的**公开文档仓库**，包含：

- **设计规范 (specs/)**：每个模块的详细设计文档
- **实施计划 (plans/)**：每个阶段的开发计划
- **架构图 (architecture/)**：执行架构、子系统分解、P0 设计
- **架构决策记录 (adr/)**：关键技术的选型原因和权衡
- **阶段汇总 (PHASES.md)**：每个阶段完成的功能
- **项目记忆 (PROJECT_MEMORY.md)**：项目全局状态和技术栈

> ⚠️ 本仓库只含设计和文档，不含代码实现（除公开的 Decision 模块结构）。完整代码见主仓库。
